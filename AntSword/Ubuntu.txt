ubuntu初始化：
	ubuntu 重启网卡
	sudo service network-manager restart

	Ubuntu 重启服务
	sudo /etc/init.d/apache2 restart

	Ubuntu ssh服务
	查看是否安装openssh-server
	dpkg -l | grep ssh

	搜索openssh-server
	sudo apt-cache search openssh-server

	安装openssh-server
	sudo apt-get install openssh-server

	查看ssh服务状态
	service sshd status
	或
	systemctl status ssh


	修改sshd_config配置文件
	vi /etc/ssh/ssh_config

	更改为以下内容：
	LoginGraceTime 2m
	PermitRootLogin yes
	StrictModes yes

	重启ssh服务
	systemctl restart ssh

	Ubuntu Vi基础用法：

	查看防火墙状态
	sudo ufw status
	
	关闭防火墙
	sudo ufw disable
	
	开启防火墙
	sudo ufw enable

	行数+gg ： 跳转到指定行，例跳转到123行：123gg
